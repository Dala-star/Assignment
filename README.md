# Optical Illusion Website Assignment

## Overview

This project is a simple web design assignment created for university coursework.  
The purpose of this assignment is to demonstrate basic knowledge of HTML structure, 
webpage layout, image integration, and content presentation.

The website displays an optical illusion artwork and provides a detailed description 
explaining how the image can be interpreted in multiple ways depending on perspective.

---

## Project Objectives

- Practice writing structured HTML
- Display and style an image on a webpage
- Create organized and readable content
- Demonstrate understanding of visual perception concepts
- Upload and manage a project using GitHub

---

## Description of the Artwork

The optical illusion used in this project combines two distinct scenes within one image.  
On one side, a woman sits reading while an elderly figure stands behind her.  
On the other side, a young man sits near a lake beside a swan.

When viewed as a whole, these two scenes merge to form the profile of a human face.  
This demonstrates how perspective and focus influence human perception.
